An Alpine.js-like framework powered by TC39 Signals

Key Features:
  • Uses TC39 Signal proposal primitives under the hood
  • Automatic dependency tracking via signals
  • Lazy evaluation with computed signals
  • Familiar Alpine.js-like directive syntax
  • ~3.6KB gzipped (compared to Alpine.js ~14KB)

1. Counter (Basic Reactivity)

<div x-data="{ count: 0 }">
  <div x-text="count"></div>
  <button @click="count++">+</button>
</div>

2. Two-way Binding (x-model)

<div x-data="{ name: 'World', greeting: () => 'Hello, ' + name + '!' }">
  <input x-model="name">
  <span x-text="greeting"></span>
</div>

How It Works

Under the hood:

  1. x-data creates Signal.State for each property
  2. Functions become Signal.Computed with automatic dependency tracking
  3. x-text, x-bind etc. create effects that update the DOM
  4. When a signal changes, only dependent effects re-run (fine-grained reactivity)

This is the same algorithm that powers Svelte 5's runes, but accessible without a compiler!
